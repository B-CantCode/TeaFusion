TROUBLESHOOTING
===============

 Python not found?
    Download from https://www.python.org/downloads/

 Module not found?
    Activate virtual environment first
    Run: .\.venv\Scripts\Activate (Windows)

 Model file missing?
    App runs in DEMO mode (simulated predictions)
    Place tea_doctor_v7_final.tflite to enable real predictions

 App won't start?
    Check requirements.txt is in same folder
    Try: streamlit cache clear
    Restart terminal
